    Class OperatePackage

    Base class for operating packages. Implement FindOperatePackages when operating on packages. Remember to change onlyStable and method.

    パッケージを操作する際の基底クラス。パッケージを操作する際は FindOperatePackages を実装。onlyStable, method の変更を忘れないように。
